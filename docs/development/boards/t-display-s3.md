## LilyGo T-Display-S3 — Справочная документация

### Чипсет и вычислительные ресурсы

**MCU:** ESP32-S3R8, Dual-core Xtensa LX7, до 240 МГц
**Flash:** 16 MB (встроенный)
**PSRAM:** 8 MB (встроенный, суффикс R8 в названии чипа означает именно это)
**Wireless:** Wi-Fi 802.11 b/g/n + Bluetooth 5.0 + BT Mesh

---

### Дисплей

| Параметр | Значение |
|---|---|
| Драйвер | ST7789V |
| Размер | 1.9" по диагонали |
| Разрешение | 170 × 320 px |
| Тип | IPS TFT LCD |
| Интерфейс | **8-Bit Parallel** (не SPI — важно!) |
| PPI | 260 |
| Цвет | Full color |

**GPIO дисплея:**
`LCD_BL=38, D0=39, D1=40, D2=41, D3=42, D4=45, D5=46, D6=47, D7=48`
`LCD_WR=08, LCD_RD=09, LCD_DC=07, LCD_CS=06, LCD_RES=05, LCD_Power_On=15`

---

### Кнопки и специальные пины

- **BOOT** — расположена на обратной стороне, рядом с RST
- **RST (Reset)**
- **IO14** — третья кнопка
- **LCD_BAT_VOLT** — GPIO04 (АЦП для измерения напряжения батареи)

---

### Pinout (из схемы)

**Левый ряд:**
`GPIO01 (ADC1_CH0, TOUCH01), GPIO02 (ADC1_CH1, TOUCH02), GPIO03 (ADC1_CH2, TOUCH03), GPIO10 (ADC1_CH9, SPI_CS, TOUCH10), GPIO11 (SPI_D, TOUCH11), GPIO12 (ADC2_CH1, SPI_CLK, TOUCH12), GPIO13 (ADC2_CH2, SPI_Q, TOUCH13)`

**Правый ряд:**
`GPIO43 (CLK_OUT1), GPIO44 (CLK_OUT2), GPIO18 (U1_RXD, ADC2_CH7), GPIO17 (U1_TXD, ADC2_CH6), GPIO21, GPIO16 (ADC2_CH5)`

**Питание:** 3V, 5V, GND (несколько пинов)

### ⚠️ Критически важные особенности

#### USB / загрузка прошивки
Плата использует **USB как JTAG-порт** (нативный USB ESP32-S3, не через отдельный CH340/CP2102).

- При использовании `Serial` через USB необходимо включить **`CDC_ON_BOOT=Enabled`** в настройках Arduino IDE
- Если порт не определяется — нужно **войти в режим загрузки вручную:**

  **Вариант A (питание отключено):**
  1. Зажать BOOT
  2. Подключить USB
  3. Отпустить BOOT

  **Вариант B (питание подключено):**
  1. Зажать BOOT
  2. Нажать и отпустить RST
  3. Отпустить BOOT

- **ESP32-S3 не поддерживает Windows 7**
- Если порт появился, но прошивка не заливается — попробовать baud rate **115200**
- Если порт не появляется совсем — сменить USB-кабель или порт
